# Installation
    1 In iTerm2, go to Preferences > Profiles and create a new profile by pressing + in bottom left corner or edit the default profile. Loacated in the init folder

    2. Select your profile and go to the Colors-tab, in the bottom right there is a dropdown with the text Color Presets.... Click it and then choose Import. Find your downloaded color scheme and import it, then select it afterwards in the dropdown.

    3. Change font to Fira Code
